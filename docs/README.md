# Dokuman Dizini

Bu klasor, algoritma sahibi ile sistem/AI entegrasyon katmanini ayirmak icin kullanilir.

## Ilk Bakilacak Dosyalar

- `algoritma_sahibi_hizli_baslangic.md`
  - sadece algoritma yazacak kisi icin en kisa akisi anlatir
- `algoritma_entegrasyon_rehberi.md`
  - sahiplik siniri, manifest mantigi ve AI entegrasyon kurallarini anlatir
- `manifest_taslak_olusturma.md`
  - JSON spec, scaffold yardimcilari ve CLI akisini anlatir

## Onerilen Okuma Sirasi

1. `algoritma_sahibi_hizli_baslangic.md`
2. `algoritma_entegrasyon_rehberi.md`
3. `manifest_taslak_olusturma.md`

## Hedef

Bu dizindeki akisin amaci su:

- sen sadece algoritmayi yazarsin
- AI manifesti ve UI baglarini ekler
- sistem yeni blogu guvenli sekilde yukler
